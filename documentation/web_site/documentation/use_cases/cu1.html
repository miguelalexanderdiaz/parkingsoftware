Documentación | Iniciando Sesión

Cuando se inicia sesión, se permite a los empleados del parqueadero relacionados con el sistema ingresar a éste mediante sus datos de sesión (Usuario y Contraseña), donde finalmente un mensaje se muestra confirmando el inicio de sesión en el sistema. Los usuarios que pueden iniciar sesión en ParQuick pueden ser el Operario y el Administrador.

Para iniciar sesión inicie la aplicación ParQuick, para ello haga clic en inicio y seguidamente haga clic en ParQuick

[IMAGEN INICIAL DE LA APLICACIÓN] - CU1.1

Ingrese su usuario en el campo de texto Usuario. Seguidamente ingrese su contraseña en el campo de texto Contraseña.

[IMÁGEN USUARIO Y CONTRASEÑA] - CU1.2

Para que el sistema verifique sus credenciales y le permita ingresar, presione la tecla Enter o haga clic en Iniciar Sesión.

[IMÁGEN BOTÓN INICIAR SESIÓN] - CU1.3

Tenga en cuenta lo siguiente:

Si los datos ingresados son incorrectos, el sistema se lo notificará y no podrá acceder a él.

Si los datos ingresados corresponden a un usuario que no existe, el sistema se lo notificará y no podrá acceder a él.

Si hace clic en el botón Manual del Usuario el sistema mostrará la sección de ayuda.

[IMÁGEN BOTÓN MANUAL DEL USUSARIO] - CU1.4
